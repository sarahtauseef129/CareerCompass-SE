CareerCompass – Sprint 1
Overview

CareerCompass is a web-based career guidance platform designed to help users discover suitable career paths based on their interests, skills, and preferences.

Sprint 1 focuses on building the core frontend experience, including career assessment, result generation, and basic interaction features.

Features Implemented (Sprint 1)
1. Career Assessment

Users can take an interactive assessment

Captures:

Interests
Skills
Preferences

2. Career Matching System

Processes user input

Generates Top 5 career recommendations

Displays:

Compatibility score (%)
Required skills
Missing skills

3. Results Dashboard

Visual representation of results:
Bar chart (career compatibility scores)
Radar chart (skill comparison)
Clean and responsive UI

4. Career Cards

Displays:

Career title
Industry
Description
Salary range
Growth outlook
Bookmark functionality added

5. Bookmark System

Users can:

Save careers
Remove saved careers
Stored locally for now. 

The bookmarked careers will be shown on a seperate page in next sprint
